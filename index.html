Manual de Personalización de StreamFusion

Importante

Esta guía te ayudará a personalizar los elementos principales de tu aplicación StreamFusion. Asegúrate de hacer una copia de seguridad de los archivos originales antes de realizar cambios.

Cambio de Idioma

Si deseas cambiar el idioma en que se muestran los títulos y la información de las películas y series, debes buscar en el código todas las instancias de es-MX y cambiarlas por el código del idioma que desees:

  • Inglés: en-US
  • Español (España): es-ES
  • Francés: fr-FR
  • Alemán: de-DE
  • Italiano: it-IT
  • Portugués (Brasil): pt-BR

Estas referencias suelen encontrarse en las llamadas a la API de TMDB (The Movie Database) que obtiene la información de películas y series.

Cambiar el Color del Título

Para cambiar el color del título "StreamFusion", modifica la clase CSS "streamfusion":

.streamfusion {
    font-weight: bold;
    background: linear-gradient(90deg, red 0%, red 50%, yellow 50%, yellow 100%);
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
}

Puedes cambiar los colores del gradiente o usar un color sólido:

/* Ejemplo con otros colores de gradiente */
.streamfusion {
    font-weight: bold;
    background: linear-gradient(90deg, #3498db 0%, #3498db 50%, #2ecc71 50%, #2ecc71 100%);
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
}

/* Ejemplo con color sólido */
.streamfusion {
    font-weight: bold;
    color: #3498db;
    -webkit-text-fill-color: #3498db;

Añadir o Quitar Categorías

Para añadir o quitar categorías, modifica la sección HTML de categorías:

<section class="categories">
    <div class="category selected" data-category="inicio">Inicio</div>
    <div class="category" data-category="peliculas">Películas</div>
    <div class="category" data-category="series">Series</div>
    <div class="category" data-category="animes">Animes</div>
    <div class="category" data-category="doramas">Doramas</div>
    <div class="category" data-category="continuar-viendo">Visto</div>
    <div class="category" data-category="TV2"><a href="go:TV2" target="_blank" style="text-decoration: none; color: inherit" rel="noreferrer">TV</a></div>
    <div class="category" data-category="Radio"><a href="go:radios" target="_blank" style="text-decoration: none; color: inherit" rel="noreferrer">Radio</a></div>
    <div class="category" data-category="peticion"><a href="go:sr" target="_blank" style="text-decoration: none; color: inherit" rel="noreferrer">S/R</a></div>

Importante: Si añades una nueva categoría, también deberás añadir la lógica JavaScript correspondiente para que funcione correctamente. Busca la sección donde se manejan los eventos de clic en las categorías.

    service.addEventListener('click', (event) => {
        const selectedService = event.currentTarget.dataset.service;
        let movieIds = [];
        let seriesIds = [];

        if (selectedService === 'netflix') {
            movieIds = [...netflixMovieIds];
            seriesIds = [...netflixSeriesIds];
        } else if (selectedService === 'disney') {
            movieIds = [...disneyMovieIds];
            seriesIds = [...disneySeriesIds];
        } else if (selectedService === 'hbo') {
            movieIds = [...hboMovieIds];
            seriesIds = [...hboSeriesIds];
        } else if (selectedService === 'prime') {
            movieIds = [...primeMovieIds];
            seriesIds = [...primeSeriesIds];
        } else if (selectedService === 'paramount') {
            movieIds = [...paramountMovieIds];
            seriesIds = [...paramountSeriesIds];
        } else if (selectedService === 'apple') { // Añadir esta condición
            movieIds = [...appleMovieIds];
            seriesIds = [...appleSeriesIds];
        }

        // Resto del código...
    });
});

Personalización de la Sección Películas

La sección de películas utiliza variables CSS para definir los colores principales. Puedes modificar estas variables para cambiar el aspecto visual:

:root {
    --background-primary: #121212;    /* Fondo principal */
    --background-secondary: #1E1E1E;  /* Fondo secundario */
    --text-primary: #FFFFFF;          /* Texto principal */
    --text-secondary: #dedede;        /* Texto secundario */
    --accent-primary: #E50914;        /* Color de acento primario (rojo) */
    --accent-secondary: #F5C518;      /* Color de acento secundario (amarillo) */
    --ui-element: #2C2C2C;            /* Elementos de UI */
    --ui-hover: #454545;              /* Estado hover de elementos UI */
}

Cambia estos valores para personalizar completamente el esquema de colores de la página de películas.

Modificar la Lista de Canales

Para añadir o quitar canales, busca el array "channels" en el código JavaScript:

const channels = [
    {
        id: "caracol-tv",
        name: "Caracol TV",
        logo: "https://upload.wikimedia.org/wikipedia/commons/thumb/5/59/Caracol_Televisi%C3%B3n_logo.svg/512px-Caracol_Televisi%C3%B3n_logo.svg.png",
        type: "hls",
        source: "https://mdstrm.com/live-stream-playlist/574463697b9817cf0886fc17.m3u8",
        categories: ["Colombia"]
    },
    {
        id: "caracol-internacional",
        name: "Caracol Internacional",
        logo: "https://upload.wikimedia.org/wikipedia/commons/a/ab/Caracol_Internacional_2023.png",
        type: "iframe",
        source: "https://embed.sdfgnksbounce.com/embed2/caracol.html",
        categories: ["Colombia"]
    },
    // Más canales...
]

Para añadir un nuevo canal, agrega un nuevo objeto al array con la siguiente estructura:

{
    id: "id-unico-del-canal", // ID único para el canal
    name: "Nombre del Canal", // Nombre que se mostrará
    logo: "URL_DEL_LOGO", // URL del logo del canal
    type: "hls", // Tipo: "hls" para streaming HLS o "iframe" para embebidos
    source: "URL_DE_LA_FUENTE", // URL del stream o iframe
    categories: ["Categoría1", "Categoría2"] // Categorías a las que pertenece
}

Para eliminar un canal, simplemente elimina el objeto correspondiente del array.

Consejos Adicionales

Recomendaciones generales para personalizar StreamFusion

  • Siempre haz una copia de seguridad de los archivos originales antes de realizar cambios.
  • Prueba los cambios en un entorno de desarrollo antes de implementarlos en producción.
  • Si añades nuevas categorías o servicios, asegúrate de actualizar también la lógica JavaScript correspondiente.
  • Para cambios más avanzados, considera consultar la documentación de las APIs utilizadas.
  • Si encuentras problemas, revisa la consola del navegador para identificar posibles errores.
  • Utiliza herramientas como el inspector de elementos del navegador para probar cambios de estilo antes de implementarlos.
  • Mantén la coherencia visual en todas las secciones para una mejor experiencia de usuario.
